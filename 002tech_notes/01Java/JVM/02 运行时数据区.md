JVM 运行时数据区包括：


- 与进程生命周期一致
    - 堆（Heap）
    - 方法区（Method Area）
    - 运行时常量池（Runtime Constant Pool）
- 与线程生命周期一致
    - JVM Stack
    - Native Method Stack
    - PC Register  


方法区：1.8之前是永久代，1.8之后是元空间。


非堆：方法区 + Code Cache


## PC 程序计数器
- 指示要运行的下一行代码的地址
- 线程私有，每个线程一个
- 执行到本地方法时，是 undefined
