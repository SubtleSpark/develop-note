### 索引上叶节点和非叶节点比较

|                  | 非叶节点（目录）      | 叶节点（数据）     |
| ---------------- | ------------- | ----------- |
| 记录项 record_type  | 1             | 0           |
| 记录项 min_rec_mask | 每页最小项为 1      | 0           |
| 记录的列数            | 只有主键和页号，没有隐藏列 | 表的所有列，包括隐藏列 |
